# ceur-poc — Confidential EUR Stablecoin (PoC)

Built on Zama's fhEVM, this ERC-7984 confidential stablecoin wraps Circle's EURC to provide encrypted balances and transfers alongside on-chain compliance primitives (KYC allowlist, freeze, force transfer, pause, observer access).

Developed as the practical artefact of a Master's thesis:
**_Development and Evaluation of a Confidential Stablecoin using fhEVM for Confidential B2B Payments._**

## Status

Phase 2 (implementation) — in progress. Detailed documentation will follow as the milestones close.
